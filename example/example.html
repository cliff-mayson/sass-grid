Grid utilities

Grid units can be hidden at any breakpoint.

visible on mobile
visible on tablet
visible on destop
                <div class="grid">
                    <div class="grid__unit" g-sm="invisible">visible on mobile </div>

                    <div class="grid__unit" g-xs="invisible" g-lg="visible">visible on destop </div>

Margins and padding

Gotch-yas

With the Inline-Block grid, the grid units' font size is reset to medium and does not inherit the font size from classes higher up. You need to apply styles.
